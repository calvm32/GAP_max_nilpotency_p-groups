# Configuring GAP

1. "**./configure && make**"
    - compile the GAP core system

2. "**cd pkg**"
    - change to the *pkg* subdirectory

3. "**../bin/BuildPackages.sh**"
    - finish configuring

# Using GAP

1. "**cd gap-4.14.0**"
    - enter the correct directory

2. "**./gap**"
    - start

3. "**Read("../AbsolutePath/../Example.g");**"
    - load source code from a file

4. "**?help** and **?sets**"
    - for help and more specific help

5. "**quit;**"
    - to exit
